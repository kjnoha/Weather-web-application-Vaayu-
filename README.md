# Vaayu — Weather Web Application

Vaayu (meaning "wind/air" in Sanskrit/Hindi) is a simple, clean weather web application built using HTML, CSS, and JavaScript. It lets users check current weather conditions for a location through a lightweight browser-based interface.

## Features

- Search weather by city/location
- Clean, responsive UI with custom background and icons
- Dedicated "About Us" page
- Built with plain HTML/CSS/JS — no frameworks required

## Project Structure

```
├── css/             # Stylesheets
├── icons/            # Weather icons used in the UI
├── js/                # JavaScript logic (API calls, DOM updates)
├── index.html        # Main application page
├── aboutus.html       # About page
├── bg.jpg            # Background image
└── cover.png          # Cover/preview image
```

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/kjnoha/Weather-web-application-Vaayu-.git
   ```
2. Open `index.html` directly in your browser, or serve it with a simple local server:
   ```bash
   npx serve .
   ```
3. Search for a city to view current weather details.

## Tech Stack

- **HTML** — structure
- **CSS** — styling and layout
- **JavaScript** — weather data fetching and dynamic UI updates
